This repository contains a collection of quantitative finance projects I built to explore the intersection of mathematics, finance, and code.

## Projects

### 1. [Efficient Frontier Visualizer](./efficient_frontier)
*   **Concept:** Modern Portfolio Theory & Diversification.
*   **Tech:** NumPy, Matplotlib.
*   **My Take:** Proving mathematically that you can lower risk without lowering returns.

### 2. [Price Derivative Analyzer](./price_derivative)
*   **Concept:** Calculus (Derivatives) applied to Momentum.
*   **Tech:** Pandas, Python.
*   **My Take:** Using 2nd derivatives (acceleration) as a leading indicator for price reversals.

### 3. [Monte Carlo Risk Simulator](./monte_carlo_sim)
*   **Concept:** Stochastic Calculus & Risk Management (VaR).
*   **Tech:** NumPy, Statistics.
*   **My Take:** Simulating 1,000 possible futures to quantify "Value at Risk".

## How to Run
1.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
2.  Navigate to any project folder and run the script.
